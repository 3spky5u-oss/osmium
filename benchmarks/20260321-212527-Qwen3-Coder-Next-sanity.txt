## 🧠 Core Principle

Binary search leverages the **sorted nature** of the input array to eliminate half of the remaining elements in each step. Instead of checking every element (as in linear search), it compares the target value to the **middle element** of the current search interval:

- If they are equal, the search is complete.
- If the target is less than the middle element, the search continues in the **left half**.
- If the target is greater, it continues in the **right half**.

This “divide-and-conquer” strategy ensures logarithmic time complexity.
